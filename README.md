# 🛒 Online Auction System  

An **Online Auction System** built with **Spring Boot (Backend)** and **React (Frontend)** that allows users to participate in live auctions, bid on products, and manage their accounts seamlessly.  

---

## ✨ Features  
- 🔐 **User Authentication** – Secure login & registration with email  
- 🛍 **Product Listing** – Sellers can list items with details & auction duration  
- ⏳ **Live Bidding** – Real-time bidding system with auto-refresh  
- 💰 **Auction Timer** – Automatic auction closure with countdown timer  
- 📜 **Bid History** – Track all past bids on an item  
- 📊 **Admin Dashboard** – Manage users, products, and auctions  
- 📩 **Email Notifications** – Alerts for bids, results, and account activity  

---

## 🏗 Tech Stack  
**Frontend:** React, Tailwind CSS  
**Backend:** Spring Boot (Java), REST API  
**Database:** MySQL  
**Build Tools:** Maven, npm  
**Server:** Tomcat  

---
